# react-native-simple-scanner

A lightweight and simple barcode/QR code scanner for React Nativee

## Installation


```sh
npm install react-native-simple-scanner
```


## Usage


```js
import { SimpleScannerView } from "react-native-simple-scanner";

// ...

<SimpleScannerView color="tomato" />
```
